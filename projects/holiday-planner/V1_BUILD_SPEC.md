# V1 Build Specification: Holiday Companion Bot

## Purpose

This file defines exactly what should be built for V1 of the Holiday Companion Bot.

V1 should be a usable local MVP, not the full final product.

## V1 Goal

Build a frontend-only local prototype that demonstrates the core Holiday Companion Bot experience.

V1 should prove that the user can:

1. Load an itinerary.
2. View today's plan.
3. View tomorrow's plan.
4. Use quick prompt chips.
5. See mock chat responses.
6. Request a proposed itinerary change.
7. Confirm or reject the change.
8. Save the updated itinerary locally.
9. See a version summary.
