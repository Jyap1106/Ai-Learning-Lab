# PRD: Second Brain

## Product Summary

Second Brain is a personal knowledge system that helps me organize my learning, career notes, AI ideas, Talent Acquisition workflows, side-hustle ideas, and future automation plans.

## Target User

Me.

## Main Problem

I collect ideas, notes, prompts, and learning materials in different places. This makes it hard to retrieve useful information later or build a future personal AI assistant.

## First Use Case

The first use case is to create a simple note structure for AI learning and Talent Acquisition automation.

## Inputs

- Learning notes
- Prompt examples
- Career reflections
- Automation ideas
- Useful resources
- Project decisions

## Outputs

- Organized notes
- Reusable prompts
- Weekly reflection logs
- Future searchable knowledge base
- Future RAG or agent-ready content

## Constraints

- Keep the structure simple
- Do not add private or sensitive information
- Do not over-engineer the system
- Start with Markdown notes before using databases or apps

## Success Criteria

This project is successful if I can store and retrieve my own learning notes, project ideas, and useful prompts without losing context.
