# Second Brain

## Project Purpose

This project explores how I can organize my personal knowledge, career notes, AI learning, finance learning, and side-hustle ideas into a system that can later become a searchable AI assistant.

## Why This Project Matters

A second brain can help me store useful information, retrieve it quickly, and eventually build a personal bot or agent that understands my goals, notes, preferences, and workflows.

## Possible Future Features

- Personal knowledge base
- Searchable notes
- Weekly review assistant
- Career decision tracker
- AI learning memory
- Side-hustle idea database
- Future RAG assistant
