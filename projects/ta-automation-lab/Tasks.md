# TASKS: TA Automation Lab

## Task 1: Create fake candidate data

Status: Not started

Goal:
Create fake candidate profiles for testing the candidate summary workflow.

Acceptance Criteria:

- Create 5 fake candidate profiles
- Use no real personal data
- Include current role, years of experience, skills, location, and notes
- Keep the data simple and easy to understand

Suggested file:

- sample-data/fake-candidates.md

Do not edit:

- README.md
- PRD.md

---

## Task 2: Create candidate summary prompt

Status: Not started

Goal:
Create a reusable prompt that turns a candidate profile into a structured recruiter summary.

Acceptance Criteria:

- Output includes summary, strengths, risks, skills match, and interview questions
- Prompt reminds the AI not to invent missing information
- Prompt uses a clear output format
- Prompt is reusable across different roles

Suggested file:

- prompts/candidate-summary.md

---

## Task 3: Test the prompt manually

Status: Not started

Goal:
Use the fake candidate data to test the candidate summary prompt manually.

Acceptance Criteria:

- Test with at least 2 fake candidates
- Save outputs in notes
- Identify what worked
- Identify what needs improvement

Suggested file:

- notes/candidate-summary-test.md
