# Cost Control Checklist

Use this before using paid AI tools, APIs, agents, or automations.

## Before using AI

- [ ] Can this be done manually first?
- [ ] Can I test with fake data?
- [ ] Can I reduce the amount of text sent to the AI?
- [ ] Can I ask the AI to read only specific files?
- [ ] Can I split the task into smaller steps?

## Before using an API

- [ ] Do I know the pricing model?
- [ ] Do I know which model I am using?
- [ ] Have I tested with 1 example first?
- [ ] Have I limited the number of rows/items processed?
- [ ] Have I set a manual stop point?
- [ ] Have I avoided infinite loops?
- [ ] Have I removed private or sensitive data?

## Before using an agent

- [ ] Does the agent have clear instructions?
- [ ] Does the agent know which files it can edit?
- [ ] Does the agent know which files it cannot edit?
- [ ] Does the agent need approval before sending, deleting, buying, or publishing anything?
- [ ] Is there a dry-run mode?

## Red Flags

Do not continue if:

- [ ] I do not understand what the automation will do
- [ ] It uses real candidate data without approval
- [ ] It can send messages automatically
- [ ] It can spend money automatically
- [ ] It can run endlessly
- [ ] It can make live trading decisions
