# Holiday Companion Bot V1

Holiday Companion Bot V1 is a frontend-only local prototype for viewing and editing a travel itinerary.

The current sample trip is a 13-day Austria itinerary.

This app is part of the larger AI Learning Lab repository.

## Current Status

The app now supports the main local V1 flow:

- View today's plan
- View tomorrow's plan
- View the full itinerary
- View a trip dashboard
- Use mock assistant prompt chips
- Receive mock itinerary-aware responses
- Generate proposed itinerary changes
- Confirm or reject proposed changes
- Save confirmed changes to localStorage
- Show local version history
- Restore previous itinerary versions
- Reset the sample trip

No real AI API is used yet.

No backend or database is used yet.
